# Boat_Runner
## About
This repository contains a project for the Computer Graphics course at Politecnico di Milano.

The project is a game in which the player must avoid the obstacles by using the keyboard to move the boat.

## 💻 Tools
🔴 C++

🔴 Vulkan API 1.3

🔴 Blender to render images and objects

## 🕹️ Navigation controls:
- Press <kbd>space</kbd> at the beginning to start the game or at the game over to restart.
- Use <kbd>w</kbd> to move the boat up.
- Use <kbd>d</kbd> to move the boat right.
- Use <kbd>a</kbd> to move the boat left.
- Use <kbd>s</kbd> to move the boat down.
